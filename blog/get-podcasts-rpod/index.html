Получаем подкасты rpod.ru с помощью python

#myworks #python #подкасты

Все-таки питон удобен. Нужно было скачать все подкасты "Сиськи-письки Шоу", пару строчек на питоне и получаем текстовый файл со ссылками, который можно скормить любому менеджеру закачек(тому же православному wget'у). Пара строчек:

1
2
3
4
5
6
7
8
9
#!/usr/bin/python
# -*- coding: utf-8 -*-
import urllib,re

page = urllib.urlopen("http://siski-piski-show.rpod.ru/rss.xml").read()
urls = re.findall("(http://.*.mp3)", page)
outfile = open("/output_file.txt","w")
for url in urls:
    outfile.write(url+"\n")

Некоторые менеджеры загрузки умеют извлекать ссылки из xml, но узнал я это уже после написания этого скрипта.
